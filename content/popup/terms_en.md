## About Ratings

To help you quickly find artwork you enjoy, each piece is displayed with a rating. We respect every artist's work — these ratings are purely subjective scores assigned by the site admin based on personal taste. They are not a measure of artistic merit and are provided for reference only. If you disagree with a rating, please feel free to disregard it.

## About Crawlers

Server resources are limited, so this site enforces rate limiting on requests. Please do not run large-scale crawlers against the site. We support digital preservation and archiving — if you need bulk original images or database access for academic or non-profit research purposes, you're welcome to contact the admin at lora@zootopia.gallery.
